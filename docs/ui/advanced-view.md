# Advanced View

## Goal

Provide an explicit home for expert and developer settings without polluting the standard workflow.

## MVP contents

- Effective settings summary
- Developer logging level
- NVTX/profiling marker toggle
- Advanced encoder overrides
- Reset advanced overrides

## Suggested leading note

> These settings override profile defaults. They are intended for testing, benchmarking, or expert tuning.

## Non-goals

- Do not move ordinary settings here.
- Do not require this page for normal recording.
